The essential WooCommerce toolkit for Bangladeshi e-commerce. Currently features Invoice & Shipping Label printing with Bulk Actions.

== Description ==

**WC4BD** is designed specifically for Bangladeshi e-commerce businesses using WooCommerce. It simplifies your order fulfillment process by providing easy-to-use tools for printing invoices and shipping labels (stickers) directly from your WooCommerce dashboard.

### Key Features
*   **Print Invoices**: Generate clean, professional invoices for your orders.
*   **Print Shipping Stickers**: Create shipping labels/stickers for your packages.
*   **Bulk Actions**: Print invoices or stickers for multiple orders at once from the order list.
*   **Admin Dashboard Integration**: Seamlessly integrated into the WooCommerce order list and order details pages.
*   **Customizable**: Add your business logo, name, address, phone number, and terms & conditions via the settings page.

This plugin is lightweight and focused on speed, ensuring it doesn't slow down your site.

**Learn more**: [Visit our landing page](https://wc4bd.commercians.com)  
**GitHub**: [View source code](https://github.com/MahmudRiazBD/WC4BD)

== Installation ==

1.  Upload the plugin files to the `/wp-content/plugins/wc4bd` directory, or install the plugin through the WordPress plugins screen directly.
2.  Activate the plugin through the 'Plugins' screen in WordPress.
3.  Go to **WooCommerce > WC4BD Settings** to configure your business details.
4.  You will now see "Print Invoice" and "Print Sticker" buttons in your WooCommerce order list and order details pages.

== Frequently Asked Questions ==

= Does this plugin work with any WooCommerce theme? =
Yes, it is designed to work with any standard WooCommerce theme.

= Can I customize the invoice design? =
Currently, you can customize the business details (logo, address, etc.). The layout is fixed for simplicity and reliability.
